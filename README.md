# Music Selector

A mobile-first Progressive Web App for music discovery and curation. Upload your YouTube Music liked songs as CSV, swipe through them Tinder-style, and export your curated selections.

## Getting Your Music Data

The CSV source file is generated by running a bookmarklet on your **YouTube Music Liked Songs** page.

### Setup

1. Go to [music.youtube.com](https://music.youtube.com) and sign in
2. Navigate to **Library → Liked Songs** (`music.youtube.com/playlist?list=LM`)
3. Create a new bookmark in your browser with the following URL (the entire block below is the bookmark URL):

```
javascript:(async()=>{const S=1500,M=9999,SEL='ytmusic-responsive-list-item-renderer';let last=-1,stable=0;for(let i=0;i<M;i++){window.scrollTo(0,document.body.scrollHeight);await new Promise(r=>setTimeout(r,S));const n=document.querySelectorAll(SEL).length;if(n===last){stable++;if(stable>=3)break}else{stable=0;last=n}}const esc=v=>`"${String(v??'').replace(/"/g,'""')}"`;const cleanUrl=u=>{try{if(!u)return'';const url=new URL(u,location.origin);url.searchParams.delete('list');return url.toString()}catch{return u||''}};const parseDuration=d=>{if(!d)return'';const p=d.split(':').map(Number);return p.length===3?p[0]*3600+p[1]*60+p[2]:p.length===2?p[0]*60+p[1]:p[0]};const rows=[...document.querySelectorAll(SEL)].map((el,i)=>{const t=el.querySelector('.title a.yt-simple-endpoint');const rawTitle=(t?.textContent||'').trim();let url=cleanUrl(t?.getAttribute('href'));let videoId='';try{videoId=new URL(url,location.origin).searchParams.get('v')||''}catch{}const cols=el.querySelectorAll('.secondary-flex-columns yt-formatted-string.flex-column');const primaryArtist=(cols[0]?.textContent||'').trim();const album=(cols[1]?.textContent||'').trim();const allArtists=primaryArtist;const duration=(el.querySelector('.fixed-column')?.textContent||'').trim();const durationSec=parseDuration(duration);const thumb=el.querySelector('img')?.src||'';const explicit=!!el.querySelector('ytmusic-inline-badge-renderer.explicit-badge');return{index:i+1,title:rawTitle,titleNormalized:rawTitle.toLowerCase(),primaryArtist,allArtists,artistCount:primaryArtist?1:0,album,duration,durationSeconds:durationSec,videoId,url,youtubeWatchUrl:videoId?`https://www.youtube.com/watch?v=${videoId}`:'',youtubeMusicUrl:videoId?`https://music.youtube.com/watch?v=${videoId}`:'',thumbnail:thumb,isExplicit:explicit}}).filter(r=>r.title);if(!rows.length)return alert('No songs found. Open music.youtube.com/library/likes first.');const header=Object.keys(rows[0]);const csv=[header.join(','),...rows.map(r=>header.map(h=>esc(r[h])).join(','))].join('\n');const a=document.createElement('a');a.href=URL.createObjectURL(new Blob([csv],{type:'text/csv;charset=utf-8'}));a.download=`ytmusic_liked_enriched_${new Date().toISOString().slice(0,10)}.csv`;a.click();setTimeout(()=>URL.revokeObjectURL(a.href),1000)})()
```

4. Click the bookmarklet while on your Liked Songs page — it will scroll through all songs and download a CSV file
5. Upload that CSV into the Music Selector app

## Features

- **CSV Upload** — Drag-and-drop or file picker with validation and preview
- **Tinder-style Swipe** — Swipe right to like, left to dislike, up to skip; with gesture and button controls
- **YouTube Playback** — Embedded player with play/pause, volume, and mini-player across all pages
- **Library** — Browse all songs with search, tab filters (All/Liked/Disliked/Unreviewed), sort, and smart grouping (by artist, album, duration, status)
- **Statistics** — Dashboard with review stats, completion tracking, distribution charts, top artists/albums
- **Export** — Export liked songs as CSV (preserving original structure), full data backup/restore as JSON
- **PWA** — Installable, works offline, service worker with caching
- **State Persistence** — All progress saved in localStorage automatically

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 19 + TypeScript |
| Build | Vite 8 |
| Styling | TailwindCSS 4 (dark theme) |
| State | Zustand + localStorage persistence |
| Animations | Framer Motion |
| CSV | PapaParse |
| PWA | vite-plugin-pwa (Workbox) |
| Routing | React Router v7 (HashRouter) |
| Icons | Lucide React |
| Deploy | GitHub Pages + GitHub Actions |
